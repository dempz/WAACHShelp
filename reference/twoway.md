# HTML summary table

**\[deprecated\]**

This function was deprecated because it was no longer required by
analysts.

## Usage

``` r
twoway(var1, var2, data = NULL, var2lab = NULL)
```

## Arguments

- var1:

  Vector of first variable

- var2:

  Vector of second variable

- data:

  Dataset containing `var1` and `var2`

- var2lab:

  Label for `var2`.

## Value

Two-way table

## Details

Two-way table similar to the "proc freq" function of SAS with two
variables
