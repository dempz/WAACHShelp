# Save dataframe

General function to save a dataset in usable formats across different
platforms. Specifically, will return .csv, .sas7bdat, and .RDS files.

## Usage

``` r
save_waachs(dataframe, path, filename)
```

## Arguments

- dataframe:

  Input dataset to save.

- path:

  Path to save file to.

- filename:

  Name of the file to save.

## Value

Saved object
