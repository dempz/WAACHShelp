# Summary tabling function

General tabling function to standardise output and formatting across
analysts and projects. Based on gtsummary::tbl_summary

## Usage

``` r
sum_tab(data, ...)
```

## Arguments

- data:

  Input dataset.

- ...:

  Any other argument relevant to
  [`gtsummary::tbl_summary`](https://www.danieldsjoberg.com/gtsummary/reference/tbl_summary.html).

## Value

Summary table.
